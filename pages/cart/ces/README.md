# 电商APP前端页面

根据提供的截图还原的电商APP前端页面，包含5个主要页面，使用专业的SVG图标系统。

## 页面列表

1. **index.html** - 商品详情页
   - 商品图片展示
   - 价格信息
   - 商品描述
   - 优惠信息
   - 底部操作栏

2. **cart.html** - 购物车页面
   - 商品列表
   - 数量调整
   - 全选功能
   - 价格计算
   - 结算按钮

3. **home.html** - 首页
   - 搜索栏
   - 分类标签
   - 商品网格展示
   - 底部导航

4. **checkout.html** - 结算页（重点功能页面）
   - 收货地址信息
   - 多店铺订单展示
   - 店铺操作（领券、优惠、去凑单）
   - 活动标签（满减、优惠券、限时抢）
   - 店铺优惠信息展示
   - 价格明细查看
   - 实时倒计时
   - 提交订单按钮

5. **弹窗功能**
   - 领券弹窗 - 展示可领取优惠券
   - 优惠弹窗 - 展示店铺所有优惠活动
   - 去凑单弹窗 - 推荐商品达到满减门槛
   - 优惠详情弹窗 - 查看当前订单优惠明细
   - 价格明细弹窗 - 查看订单价格构成

## 技术亮点

### SVG图标系统
- 所有图标使用SVG symbol定义
- 通过`<use>`标签引用，便于维护
- 支持颜色自定义（currentColor）
- 矢量图形，任意缩放不失真

### 样式兼容性
- 使用`-webkit-tap-highlight-color`优化移动端点击
- 添加`-webkit-font-smoothing`优化字体渲染
- 使用`accent-color`自定义复选框颜色
- 添加`:active`伪类提供触摸反馈
- 支持iOS Safari、Chrome、Firefox等主流浏览器

### 响应式设计
- 固定最大宽度375px，居中显示
- 使用flexbox布局
- 触摸友好的按钮尺寸
- 防止横向滚动

## 使用方法

1. 直接在浏览器中打开任意HTML文件
2. 推荐从 `home.html` 开始浏览
3. 所有页面已链接，可以相互跳转
4. 最佳体验：使用浏览器开发者工具切换到移动设备视图（375x667）

## 结算页详细功能说明

### 店铺操作
1. **领券** - 点击后弹出该店铺可领取的优惠券列表

2. **优惠** - 点击后展示店铺所有优惠活动

   - 限时优惠信息

   - 活动规则说明

3. **去凑单** - 智能推荐商品

   - 一键加购功能

- 每个店铺下方显示当前可用优惠摘要

- 包含当前可用优惠和可领取优惠

### 价格明细
- 点击底部"明细"按钮查看
- 分店铺显示商品金额
- 展示运费信息
- 列出所有优惠项目
- 计算实付金额

## 开发说明

### 修改图标
所有SVG图标定义在每个HTML文件的顶部`<div style="display: none;">`中，可以根据需要添加或修改图标。

### 修改颜色
图标颜色通过CSS的`color`属性控制，使用`currentColor`继承父元素颜色。

### 添加新页面
1. 复制现有HTML文件
2. 确保包含SVG图标定义
3. 在`script.js`中添加对应的导航逻辑

### 自定义弹窗
在`checkout.js`中使用`createModal(title, content)`函数创建新弹窗：
```javascript
const modal = createModal('标题', `
    <div class="modal-content-scroll">
        <!-- 弹窗内容 -->
    </div>
`);
```

### 添加新的活动标签
在CSS中添加新的标签样式：
```css
.tag-newtype {
    background: #颜色;
    color: #文字颜色;
    border: 1px solid #边框颜色;
}
```
